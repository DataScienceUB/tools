La **recursió** $f(x)^2 = u$ consisteix en què un programa es crida a si mateix. Un programa recursiu ben fet ha de complir 2 condicions: + ha d'haver-hi un o més casos base: són condicions de les dades que es resolen directament, sense que calgui una nova crida al programa + a cada crida recursiva ens apropem a un dels casos base: ja sigui perquè es decrementa una variable, perquè es redueix una llista... D'una manera simplificada podríem dir que els algorismes recursius segueixen el patró: ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~ def recursiu(x): if x=cas_final: return VALOR else: return CRIDA_RECURSIVA(x reduïda)+Calculs ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~ !!! Error: Compte amb la recusió. No feu mai més de 100 crides recursives d'un programa! $$f(x) = 23$$ ## Invoking it
